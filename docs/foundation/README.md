# docs/foundation/

Documentos permanentes da Foundation Empresarial do ParaguAI. Esta pasta contém os princípios, filosofias e processos que definem quem o ParaguAI é e como opera — independentemente de tecnologia, mercado ou circunstância.

## Propósito

Preservar o conhecimento permanente da empresa em documentos versionados, auditáveis e consultáveis por qualquer pessoa ou agente de IA em qualquer momento.

## Quando criar um novo documento aqui

Nunca — sem aprovação explícita do CTO. A Foundation é um conjunto fechado de princípios permanentes. Novos documentos nesta pasta representam uma expansão do núcleo filosófico da empresa, não uma adição operacional.

Se você está pensando em criar um documento aqui, provavelmente ele pertence a outra categoria (`operations/`, `engineering/`, `architecture/`). Consulte `FOUNDATION_INDEX.md` para orientação.

## Quando não criar

- Documentação técnica → `architecture/` ou `engineering/`
- Estado atual do projeto → `operations/`
- Funcionalidades e roadmap → `product/`
- Decisões pontuais → `operations/DECISIONS.md` (ADR)

## Regra de versionamento

Qualquer alteração em documento desta pasta exige:
1. Incremento de versão (ex: v1.2 → v1.3)
2. Registro no `docs/operations/CHANGELOG.md`
3. Atualização da tabela de versões acima

Nunca editar silenciosamente. Circunstâncias não sobrescrevem princípios.
